# Linear Regression from Scratch

Реализация линейной регрессии с нуля на Python без использования готовых ML-библиотек.

## 📌 О проекте

В этом проекте реализована модель линейной регрессии:

y = w * x + b

Параметры w и b обучаются с помощью градиентного спуска для минимизации функции ошибки (MSE).

---

## ⚙️ Что реализовано

- Линейная модель: y = w * x + b
- Функция ошибки (Mean Squared Error)
- Градиентный спуск
- Нормализация данных
- Предсказание на новых данных

---

## 🧩 Как работает модель

1. Данные нормализуются (приводятся к диапазону ~[0,1])
2. Инициализируются параметры w и b
3. На каждой итерации:
   - считается предсказание: y = w * x + b
   - считается ошибка (MSE)
   - вычисляются градиенты по w и b
   - параметры обновляются через градиентный спуск
4. После обучения модель используется для предсказания новых значений
---

## 🧠 Что я понял в процессе

- Модель — это просто функция, которая делает предсказание  
- Обучение — это подбор параметров через минимизацию ошибки  
- Градиент показывает направление изменения параметров  
- Learning rate (alpha) влияет на скорость и стабильность обучения  
- Без нормализации градиентный спуск работает значительно хуже  
- При уменьшении alpha требуется больше итераций для сходимости
- Слишком маленький alpha замедляет обучение без значительного прироста точности
- Модель должна обучаться отдельно от этапа предсказания
- Для предсказания восстанавливается масштаб
---
## 🔍 Влияние learning rate

- Большой alpha → нестабильное обучение  
- Маленький alpha → медленная сходимость  
- Оптимальное значение: 0.01
